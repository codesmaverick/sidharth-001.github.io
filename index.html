Password Generator

Generates random passwords with toggling between uppercase & lowercase letters, numbers and symbols. (HTML, CSS, JS). It Generates random password with following pattern scheme, a uppercase letter, a lowercase letter, a number, a symbol and repeats pattern for required length, then all the elements of password are swapped with other random elements.
